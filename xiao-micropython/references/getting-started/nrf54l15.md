# MicroPython on XIAO nRF54L15 Sense

## Overview

Getting started with MicroPython on SeeedStudio XIAO nRF54L15 Sense.

Seeed’s documentation for XIAO nRF54L15 Sense uses a workflow based on Seeed’s firmware package plus the helper modules from `micropython-seeed-boards`.

## Pin Mapping

Pin mapping is based on this repository's Seeed XIAO pin reference table.

| XIAO Pin | Chip Pin | Notes |
|----------|----------|-------|
| D0 | P1.04 | GPIO, ADC |
| D1 | P1.05 | GPIO, ADC |
| D2 | P1.06 | GPIO, ADC |
| D3 | P1.07 | GPIO, ADC |
| D4 | P1.10 | I2C-0 SDA (XIAO standard) |
| D5 | P1.11 | I2C-0 SCL (XIAO standard) |
| D6 | P2.08 | UART TX (XIAO standard) |
| D7 | P2.07 | UART RX (XIAO standard) |
| D8 | P2.01 | SPI SCK (XIAO standard) |
| D9 | P2.04 | SPI MISO (XIAO standard) |
| D10 | P2.02 | SPI MOSI (XIAO standard) |
| D11 | P0.03 | I2C-1 SCL |
| D12 | P0.04 | I2C-1 SDA |
| D13 | P2.10 | GPIO |
| D14 | P2.09 | GPIO |
| D15 | P2.06 | GPIO |

Other key pins (from Seeed Ref pin map):

- **USER_LED**: P2.00
- **USER KEY**: P0.00
- **AIN7_VBAT**: P1.14
- **NFC1/NFC2**: P1.02 / P1.03
- **RF Switch Port Select / Power**: P2.05 / P2.03
- **nRF54L15 SWD**: SWDCLK / SWDIO / RST
- **SAMD11 SWD**: PA30 / PA31 / RST2

## Setup (Seeed workflow)

1. Flash the XIAO nRF54L15 MicroPython firmware (Seeed firmware package).
2. Clone Seeed’s helper repository:

```bash
git clone https://github.com/Seeed-Studio/micropython-seeed-boards.git
```

3. Upload the `boards/` folder from `micropython-seeed-boards/examples` to the device filesystem (Thonny file manager).

## Basic Examples

### Blink User LED

```python
import time
from boards.xiao import XiaoPin

led = XiaoPin("led", XiaoPin.OUT)

while True:
    led.value(1)
    time.sleep(0.5)
    led.value(0)
    time.sleep(0.5)
```

### I2C (XIAO standard: D4/D5)

```python
from boards.xiao import XiaoI2C

sda = 4  # D4
scl = 5  # D5
bus = "i2c0"

i2c = XiaoI2C(bus, sda, scl, 400_000)
print(i2c.scan())
```

### SPI (XIAO standard: D8/D9/D10)

```python
from boards.xiao import XiaoSPI

# D8=SCK, D9=MISO, D10=MOSI
spi = XiaoSPI("spi0", 1_000_000, 8, 10, 9)

# Write only
spi.write(b"hello")
```

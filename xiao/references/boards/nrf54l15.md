# XIAO nRF54L15 Sense Reference

**Chipset**: nRF54L15 (ARM Cortex-M33 64MHz)
**Wireless**: BLE 5.4
**Flash**: 512KB
**RAM**: 128KB

## I2C

- **I2C-0**: D4 (SDA), D5 (SCL)
- **I2C-1**: D12 (SDA), D11 (SCL)

## SPI

- **SCK**: D8 (P2.01)
- **MISO**: D9 (P2.04)
- **MOSI**: D10 (P2.02)
- **CS**: Any available GPIO

## UART

- **TX**: D6 (P2.08)
- **RX**: D7 (P2.07)

## ADC

- Multiple ADC channels on P1.04-P1.07
- 12-bit resolution (0-4095)
- Range: 0-3.3V

## Sense Features

- Built-in temperature sensor
- Built-in humidity sensor
- Ultra-low power operation

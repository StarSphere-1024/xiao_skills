# XIAO RP2350 Reference

**Chipset**: RP2350 (ARM Cortex-M33 dual-core + RISC-V dual-core 150MHz)
**Wireless**: None (PIO available)
**Flash**: 4MB (external)
**RAM**: 512KB SRAM

## Pin Definitions

| Pin | GPIO | Arduino | Functions | Notes |
|-----|------|---------|-----------|-------|
| D0 | GPIO26 | D0 | A0 | ADC |
| D1 | GPIO27 | D1 | A1 | ADC |
| D2 | GPIO28 | D2 | A2 | ADC |
| D3 | GPIO5 | D3 | SPI0_CSn | SPI CS |
| D4 | GPIO6 | D4 | SDA1 | I2C1 SDA (XIAO standard) |
| D5 | GPIO7 | D5 | SCL1 | I2C1 SCL (XIAO standard) |
| D6 | GPIO0 | D6 | TX0 | UART0 TX (XIAO standard) |
| D7 | GPIO1 | D7 | RX0 | UART0 RX (XIAO standard) |
| D8 | GPIO2 | D8 | SPI0_SCK | SPI SCK (XIAO standard) |
| D9 | GPIO4 | D9 | SPI0_MISO | SPI MISO (XIAO standard) |
| D10 | GPIO3 | D10 | SPI0_MOSI | SPI MOSI (XIAO standard) |
| D11 | GPIO21 | D11 | RX1 | UART1 RX |
| D12 | GPIO20 | D12 | TX1 | UART1 TX |
| D13 | GPIO17 | D13 | SCL0 | I2C0 SCL |
| D14 | GPIO16 | D14 | SDA0 | I2C0 SDA |
| D15 | GPIO11 | D15 | SPI1_MOSI | SPI1 MOSI |
| D16 | GPIO12 | D16 | SPI1_MISO | SPI1 MISO |
| D17 | GPIO10 | D17 | SPI1_SCK | SPI1 SCK |
| D18 | GPIO9 | D18 | SPI1_CSn | SPI1 CS |

### Other Board Pins

- **ADC_BAT**: GPIO29 (battery voltage sense)
- **RESET**: RUN
- **BOOT**: RP2040_BOOT
- **RGB LED**: GPIO22
- **USER_LED / LED_BUILTIN**: GPIO25

## I2C

- **SDA0**: GPIO16 (D14)
- **SCL0**: GPIO17 (D13)
- **SDA1**: GPIO6 (D4)
- **SCL1**: GPIO7 (D5)

## SPI

- **SPI0**: D8 (SCK), D9 (MISO), D10 (MOSI), D3 (CS)
- **SPI1**: D17 (SCK), D12 (MISO), D11 (MOSI), D15 (CS)

## UART

- **UART0**: D6 (TX), D7 (RX)
- **UART1**: D20 (TX), D21 (RX)

## ADC

- 4 channels: A0-A3 (GPIO26-29)
- 12-bit resolution (0-4095)
- Range: 0-3.3V

## Key Features

- Dual ARM Cortex-M33 OR dual RISC-V cores (selectable)
- Enhanced PIO with more features
- More RAM than RP2040
- Fast SPI flash support

# XIAO RP2040 Reference

**Chipset**: RP2040 (ARM Cortex-M0+ dual-core 133MHz)
**Wireless**: None (but has PIO for custom protocols)
**Flash**: 2MB (external)
**RAM**: 264KB SRAM

## Pin Definitions

| Pin | GPIO | Arduino | Functions | Notes |
|-----|------|---------|-----------|-------|
| D0 | GPIO26 | D0 | ADC0 | Analog input |
| D1 | GPIO27 | D1 | ADC1 | Analog input |
| D2 | GPIO28 | D2 | ADC2 | Analog input |
| D3 | GPIO29 | D3 | ADC3 | Analog input |
| D4 | GPIO6 | D4 | I2C SDA | Default I2C SDA (XIAO standard) |
| D5 | GPIO7 | D5 | I2C SCL | Default I2C SCL (XIAO standard) |
| D6 | GPIO0 | D6 | UART TX | Default UART TX (XIAO standard) |
| D7 | GPIO1 | D7 | UART RX | Default UART RX (XIAO standard) |
| D8 | GPIO2 | D8 | SPI SCK | Default SPI SCK (XIAO standard) |
| D9 | GPIO4 | D9 | SPI MISO | Default SPI MISO (XIAO standard) |
| D10 | GPIO3 | D10 | SPI MOSI | Default SPI MOSI (XIAO standard) |

## XIAO Standard Peripheral Mapping

All XIAO boards follow a unified peripheral pin mapping for expansion board compatibility:

| Interface | Pin | GPIO |
|-----------|-----|------|
| **I2C SDA** | D4 | GPIO6 |
| **I2C SCL** | D5 | GPIO7 |
| **UART TX** | D6 | GPIO0 |
| **UART RX** | D7 | GPIO1 |
| **SPI SCK** | D8 | GPIO2 |
| **SPI MISO** | D9 | GPIO4 |
| **SPI MOSI** | D10 | GPIO3 |

## I2C

```cpp
#include <Wire.h>

void setup() {
    Wire.begin();  // Uses D4 (SDA) and D5 (SCL) by default
}
```

## SPI

```cpp
#include <SPI.h>

void setup() {
    SPI.begin();  // Uses D8 (SCK), D9 (MISO), D10 (MOSI) by default
}
```

## UART

```cpp
#define TX_PIN D6  // GPIO0
#define RX_PIN D7  // GPIO1

void setup() {
    Serial1.begin(115200);
}
```

## ADC

- 4 channels: A0 (GPIO26), A1 (GPIO27), A2 (GPIO28), A3 (GPIO29)
- 12-bit resolution (0-4095)
- Range: 0-3.3V

## PWM

All GPIO pins support PWM (8 slices, 2 channels each, 16 channels total)

## PIO (Programmable I/O)

- 2 PIO blocks
- 4 state machines each
- Custom protocol implementation
- Useful for: WS2812 LEDs, DHT11, custom protocols

## Bootloader (BOOTSEL)

1. Hold BOOTSEL button while connecting USB
2. Drive appears as RPI-RP2 disk
3. Drag and drop .uf2 file
4. Board reboots automatically

## SWD Debug

- **SWCLK**: GPIO24
- **SWDIO**: GPIO25

## Power

- **Input**: 5V via USB
- **Operating Voltage**: 3.3V
- **Active**: ~25mA (both cores)

## Important Notes

- No wireless hardware
- PIO can simulate many wireless protocols at low speed
- USB 1.1 device/host support
